使用说明：公式

1. 单元格运算
  • 基础运算：加(+)、减(-)、乘(*)、除(/)、取余(%)
2. 公式运算，目前支持以下方式：
  • `sum` 求和
  • `avg` 平均数
  • `max` 最大值
  • `count` 计数

书写方式(以sum为例)：

  • sum(A1,A2,B1) 等价于 A1+A2+B1
  • sum(A1 : B2) 等价于 A1 + A2 + B1 + B2 区域运算
  • sum(A1+) 等价于 A1 + A2 + A3 + A4 + ... + An 计算A1所在列，从A1开始的下方所有单元格的值
  • sum(A1-) 等价于 A1 + B1 + C1 + ... + N1 计算A1所在行，从A1开始右方所有的值
