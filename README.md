#JRelax-BI

dist目录中的zip包 解压后直接丢入tomcat
db.sql导入数据库中（Mysql数据库）

* 技术支持：QQ群490249408
* 当前版本号为：1.0

## 功能清单

* 自定义表单
* 数据库管理工具
* 自定义数据源
* 自定义工作流（基于activiti，包含web版本的流程设计器）
* 自定义图表（基于chart.js）

## 下一版本计划

* 自定义表格
* 自定义图表增加钻取功能
* 自定义表单增加动态数据源控件
